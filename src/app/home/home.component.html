This is the personal website of Nicholas J Coats and this site is mostly web experiments. I like the browser as a rich, interactive user-agent. It has its own storage, session mechanism, and a lot of adaptability built in. I have built a few small games with some different technologies, only some of which are previewed here.

Check out my code; if you dare.

This site is powered by various libraries including:

  1. PixiJS
  2. ThreeJS
  3. ParticlesJS
  4. p2.js
  5. CreateJS
  6. Angular (used to be VueJS)
The entire site is written in browser-compatible Javascript, or some language varient like Typescript. I try to write in browser-native es8+ to keep as compatible as possible without the use of a transpiled language. I don't like having the library dependency as long-term tools are still changing.
